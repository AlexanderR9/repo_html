1. установка в Ubuntu/Debian
    - apt update
    - apt install nodejs
    - apt install npm
  после этого nodejs готов к работе

2. создание проекта (под своим пользователем)
    - создать папку проекта в произвольном месте
    - войти в эту папку в терминале
    - создать главный скрипт(файл), например index.js
    - загрузить сторонние модули, необходимы для разработки вашего проекта:
	например: npm i axios
	после чего появится папка node_modules, в которую будут выкачиваться запрошенные модули(библиотеки),
	'node_modules' - папку именно с таким именем генерит nodejs, ее нельзя менять.
	а так же появиться файл(или допишется) package.json , в него добавится информация о новом модуле.

	примечание: можно заранее подготовить файл package.json, в котором заранее указать все необходимые библиотеки и их версии
	    пример:
 "dependencies": {
    "@uniswap/sdk-core": "^5.9.0",
    "@uniswap/v3-sdk": "^3.18.1",
    "axios": "^1.7.5",
    "ethers": "^5.7.2"
  }
	затем просто выполнить команду: npm i
	и все указанные пакеты в секции "dependencies" скачаются в папку node_modules


3. для работы с быблиотекой ethers если заготовленный(в архивах) набор файлов ABI и библиотек.
    - архив abi.tar.gz распаковать рядом с архивами
    - распаковать рядом с архивами создать папку 'node_modules'
    - в 'node_modules' распаковать dotenv.tar.gz
    - в 'node_modules' распаковать содержимое ethers.tar.gz/eth_kit
    после чего можно выполнять скрипты, в которых подключена 'ethers'
